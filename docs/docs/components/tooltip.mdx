# Tooltip 文字提示

简单的文字提示气泡框。

## 基本用法

```jsx live
function TooltipExample() {
  return (
    <div style={{ padding: '20px', display: 'flex', gap: '16px', alignItems: 'center' }}>
      <Tooltip title="这是一个提示信息">
        <Button>鼠标悬停显示提示</Button>
      </Tooltip>
      
      <Tooltip title="这是一个很长的提示信息，用来演示文字提示的换行效果">
        <Button>长文本提示</Button>
      </Tooltip>
    </div>
  );
}
```

## 受控模式

```jsx live
function TooltipControlledExample() {
  const [open, setOpen] = React.useState(false);
  
  return (
    <div style={{ padding: '20px' }}>
      <div style={{ marginBottom: '16px', display: 'flex', gap: '8px' }}>
        <Button onClick={() => setOpen(true)}>显示</Button>
        <Button onClick={() => setOpen(false)}>隐藏</Button>
        <Button onClick={() => setOpen(!open)}>切换</Button>
      </div>
      
      <Tooltip 
        title="受控的提示信息" 
        open={open}
        onOpenChange={setOpen}
      >
        <Button>受控提示</Button>
      </Tooltip>
    </div>
  );
}
```

## API

### Tooltip Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| arrow | 修改箭头的显示状态以及修改箭头是否指向目标元素的中心 | `boolean \| { pointAtCenter: boolean }` | `true` |
| autoAdjustOverflow | 气泡被遮挡时自动调整位置 | `boolean` | `true` |
| color | 背景颜色 | `string` | - |
| defaultOpen | 默认是否显隐 | `boolean` | `false` |
| destroyTooltipOnHide | 关闭后是否销毁 Tooltip | `boolean` | `false` |
| fresh | 默认情况下，Tooltip 在关闭时会缓存内容。设置该属性后会始终保持更新 | `boolean` | `false` |
| getPopupContainer | 浮层渲染父节点，默认渲染到 body 上 | `(triggerNode: HTMLElement) => HTMLElement` | `() => document.body` |
| mouseEnterDelay | 鼠标移入后延时多少才显示 Tooltip，单位：秒 | `number` | `0.1` |
| mouseLeaveDelay | 鼠标移出后延时多少才隐藏 Tooltip，单位：秒 | `number` | `0.1` |
| overlayClassName | 卡片类名 | `string` | - |
| overlayStyle | 卡片样式 | `object` | - |
| overlayInnerStyle | 卡片内容区域的样式对象 | `object` | - |
| placement | 气泡框位置 | `string` | `'top'` |
| trigger | 触发行为 | `'hover' \| 'focus' \| 'click' \| 'contextMenu' \| Array<string>` | `'hover'` |
| open | 用于手动控制浮层显隐 | `boolean` | - |
| zIndex | 设置 Tooltip 的 z-index | `number` | - |
| onOpenChange | 显示隐藏的回调 | `(open: boolean) => void` | - |
| title | 提示文字 | `ReactNode \| () => ReactNode` | - |
| className | 自定义类名 | `string` | - |
| style | 自定义样式 | `React.CSSProperties` | - |
| disabled | 是否禁用 | `boolean` | `false` |

### 注意

请确保 `Tooltip` 的子元素能接受 `onMouseEnter`、`onMouseLeave`、`onFocus`、`onClick` 事件。

### 常用位置

| 位置 | 说明 |
| --- | --- |
| `top` | 上边，居中对齐 |
| `topLeft` | 上边，左对齐 |
| `topRight` | 上边，右对齐 |
| `bottom` | 下边，居中对齐 |
| `bottomLeft` | 下边，左对齐 |
| `bottomRight` | 下边，右对齐 |
| `left` | 左边，居中对齐 |
| `leftTop` | 左边，上对齐 |
| `leftBottom` | 左边，下对齐 |
| `right` | 右边，居中对齐 |
| `rightTop` | 右边，上对齐 |
| `rightBottom` | 右边，下对齐 |
