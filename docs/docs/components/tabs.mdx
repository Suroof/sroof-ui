# Tabs 标签页

标签页组件，支持受控和非受控两种模式，提供灵活的标签页导航功能。

## 基本用法

```jsx live
function TabsExample() {
  return (
    <Tabs defaultActiveKey="tab1">
      <Tab key="tab1" label="标签页 1">
        <div style={{ padding: '20px' }}>
          <h3>标签页 1 内容</h3>
          <p>这是第一个标签页的内容。</p>
        </div>
      </Tab>
      <Tab key="tab2" label="标签页 2">
        <div style={{ padding: '20px' }}>
          <h3>标签页 2 内容</h3>

        </div>
      </Tab>
      <Tab key="tab3" label="标签页 3">
        <div style={{ padding: '20px' }}>

          <p>这是第三个标签页的内容。</p>
        </div>

    </Tabs>
  );
}

## 动态标签页

```jsx live
function DynamicTabsExample() {
  const [tabs, setTabs] = React.useState([
    { key: 'tab1', label: '标签 1', content: '这是标签 1 的内容' },
    { key: 'tab2', label: '标签 2', content: '这是标签 2 的内容' }
  ]);
  const [activeKey, setActiveKey] = React.useState('tab1');
  const [counter, setCounter] = React.useState(3);

  const addTab = () => {
    const newTab = {
      key: `tab${counter}`,
      label: `标签 ${counter}`,
      content: `这是动态添加的标签 ${counter} 的内容`
    };
    setTabs([...tabs, newTab]);
    setActiveKey(newTab.key);
    setCounter(counter + 1);
  };

  const removeTab = (targetKey) => {
    const newTabs = tabs.filter(tab => tab.key !== targetKey);
    setTabs(newTabs);
    
    if (activeKey === targetKey && newTabs.length > 0) {
      setActiveKey(newTabs[0].key);
    }
  };

  return (
    <div>
      <div style={{ marginBottom: '16px' }}>
        <button 
          onClick={addTab}
          style={{ 
            padding: '8px 16px', 
            border: 'none', 
            borderRadius: '4px', 
            background: '#10b981', 
            color: 'white',
            cursor: 'pointer',
            marginRight: '8px'
          }}
        >
          ➕ 添加标签页
        </button>
        <span style={{ color: '#6b7280' }}>
          当前有 {tabs.length} 个标签页
        </span>
      </div>

      <Tabs
        activeKey={activeKey}
        onChange={setActiveKey}
      >
        {tabs.map(tab => (
          <Tab 
            key={tab.key} 
            label={
              <span style={{ display: 'flex', alignItems: 'center', gap: '8px' }}>
                {tab.label}
                {tabs.length > 1 && (
                  <button
                    onClick={(e) => {
                      e.stopPropagation();
                      removeTab(tab.key);
                    }}
                    style={{
                      background: 'none',
                      border: 'none',
                      color: '#ef4444',
                      cursor: 'pointer',
                      padding: '2px',
                      borderRadius: '2px',
                      fontSize: '12px'
                    }}
                    title="关闭标签页"
                  >
                    ✕
                  </button>
                )}
              </span>
            }
          >
            <div style={{ padding: '20px' }}>
              <h3>{tab.label}</h3>
              <p>{tab.content}</p>
              <div style={{ marginTop: '16px', color: '#6b7280', fontSize: '14px' }}>
                标签页 ID: {tab.key}
              </div>
            </div>
          </Tab>
        ))}
      </Tabs>
    </div>
  );
}
```

```jsx
function DynamicTabs() {
  const [tabs, setTabs] = useState([
    { key: 'tab1', label: '标签 1', content: '内容 1' }
  ]);
  const [activeKey, setActiveKey] = useState('tab1');

  const addTab = () => {
    const newKey = `tab${Date.now()}`;
    setTabs([...tabs, { 
      key: newKey, 
      label: `新标签 ${tabs.length + 1}`, 
      content: `新内容 ${tabs.length + 1}` 
    }]);
    setActiveKey(newKey);
  };

  const removeTab = (targetKey) => {
    const newTabs = tabs.filter(tab => tab.key !== targetKey);
    setTabs(newTabs);
    if (activeKey === targetKey && newTabs.length > 0) {
      setActiveKey(newTabs[0].key);
    }
  };

  return (
    <div>
      <button onClick={addTab}>添加标签页</button>
      <Tabs activeKey={activeKey} onChange={setActiveKey}>
        {tabs.map(tab => (
          <Tab key={tab.key} label={tab.label}>
            <div>{tab.content}</div>
          </Tab>
        ))}
      </Tabs>
    </div>
  );
}
```

## API

### Tabs Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| activeKey | `string` | - | 当前激活的标签页 key（受控模式） |
| children | `React.ReactNode` | - | Tab 子组件 |
| centered | `boolean` | `false` | 标签页是否居中显示 |
| className | `string` | - | 自定义样式类名 |
| defaultActiveKey | `string` | - | 默认激活的标签页 key（非受控模式） |
| onChange | `(key: string) => void` | - | 标签页切换时的回调函数 |

### Tab Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| key | `string` | - | 标签页的唯一标识符（必填） |
| label | `React.ReactNode` | - | 标签页显示的标签文本或元素 |
| children | `React.ReactNode` | - | 标签页的内容 |

## 注意事项

1. **Key 的重要性**: 每个 `Tab` 必须有唯一的 `key` 属性
2. **受控 vs 非受控**: 使用 `activeKey` 为受控模式，使用 `defaultActiveKey` 为非受控模式
3. **内容渲染**: 只有当前激活的标签页内容会被渲染
4. **键盘导航**: 支持左右箭头键在标签间导航
5. **无障碍访问**: 自动生成正确的 ARIA 属性和角色

## 最佳实践

1. **标签数量**: 建议标签页数量不超过 7 个，过多会影响用户体验
2. **标签文本**: 保持标签文本简洁明了，避免过长
3. **内容组织**: 确保标签页之间的内容逻辑清晰
4. **响应式**: 在移动端考虑标签页的显示方式
5. **状态管理**: 合理使用受控和非受控模式

## 动态标签页

```jsx live
function DynamicTabsExample() {
  const [tabs, setTabs] = React.useState([
    { key: 'tab1', label: '标签 1', content: '这是标签 1 的内容' },
    { key: 'tab2', label: '标签 2', content: '这是标签 2 的内容' }
  ]);
  const [activeKey, setActiveKey] = React.useState('tab1');
  const [counter, setCounter] = React.useState(3);

  const addTab = () => {
    const newTab = {
      key: `tab${counter}`,
      label: `标签 ${counter}`,
      content: `这是动态添加的标签 ${counter} 的内容`
    };
    setTabs([...tabs, newTab]);
    setActiveKey(newTab.key);
    setCounter(counter + 1);
  };

  const removeTab = (targetKey) => {
    const newTabs = tabs.filter(tab => tab.key !== targetKey);
    setTabs(newTabs);
    
    if (activeKey === targetKey && newTabs.length > 0) {
      setActiveKey(newTabs[0].key);
    }
  };

  return (
    <div>
      <div style={{ marginBottom: '16px' }}>
        <button 
          onClick={addTab}
          style={{ 
            padding: '8px 16px', 
            border: 'none', 
            borderRadius: '4px', 
            background: '#10b981', 
            color: 'white',
            cursor: 'pointer',
            marginRight: '8px'
          }}
        >
          ➕ 添加标签页
        </button>
        <span style={{ color: '#6b7280' }}>
          当前有 {tabs.length} 个标签页
        </span>
      </div>

      <Tabs
        activeKey={activeKey}
        onChange={setActiveKey}
      >
        {tabs.map(tab => (
          <Tab 
            key={tab.key} 
            label={
              <span style={{ display: 'flex', alignItems: 'center', gap: '8px' }}>
                {tab.label}
                {tabs.length > 1 && (
                  <button
                    onClick={(e) => {
                      e.stopPropagation();
                      removeTab(tab.key);
                    }}
                    style={{
                      background: 'none',
                      border: 'none',
                      color: '#ef4444',
                      cursor: 'pointer',
                      padding: '2px',
                      borderRadius: '2px',
                      fontSize: '12px'

function DynamicTabs() {
  const [tabs, setTabs] = useState([
    { key: 'tab1', label: '标签 1', content: '内容 1' }
  ]);
  const [activeKey, setActiveKey] = useState('tab1');

  const addTab = () => {
    const newKey = `tab${Date.now()}`;
    setTabs([...tabs, { 
      key: newKey, 
      label: `新标签 ${tabs.length + 1}`, 
      content: `新内容 ${tabs.length + 1}` 
    }]);
    setActiveKey(newKey);
  };

  const removeTab = (targetKey) => {
    const newTabs = tabs.filter(tab => tab.key !== targetKey);
    setTabs(newTabs);
    if (activeKey === targetKey && newTabs.length > 0) {
      setActiveKey(newTabs[0].key);
    }
  };

  return (
    <div>
      <button onClick={addTab}>添加标签页</button>
      <Tabs activeKey={activeKey} onChange={setActiveKey}>
        {tabs.map(tab => (
          <Tab key={tab.key} label={tab.label}>
            <div>{tab.content}</div>
          </Tab>
        ))}
      </Tabs>
    </div>
  );
}

## API

### Tabs Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| activeKey | `string` | - | 当前激活的标签页 key（受控模式） |
| children | `React.ReactNode` | - | Tab 子组件 |
| centered | `boolean` | `false` | 标签页是否居中显示 |
| className | `string` | - | 自定义样式类名 |
| defaultActiveKey | `string` | - | 默认激活的标签页 key（非受控模式） |
| onChange | `(key: string) => void` | - | 标签页切换时的回调函数 |

### Tab Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| key | `string` | - | 标签页的唯一标识符（必填） |
| label | `React.ReactNode` | - | 标签页显示的标签文本或元素 |
| children | `React.ReactNode` | - | 标签页的内容 |

## 注意事项

1. **Key 的重要性**: 每个 `Tab` 必须有唯一的 `key` 属性
2. **受控 vs 非受控**: 使用 `activeKey` 为受控模式，使用 `defaultActiveKey` 为非受控模式
3. **内容渲染**: 只有当前激活的标签页内容会被渲染
4. **键盘导航**: 支持左右箭头键在标签间导航
5. **无障碍访问**: 自动生成正确的 ARIA 属性和角色

## 最佳实践

1. **标签数量**: 建议标签页数量不超过 7 个，过多会影响用户体验
2. **标签文本**: 保持标签文本简洁明了，避免过长
3. **内容组织**: 确保标签页之间的内容逻辑清晰
4. **响应式**: 在移动端考虑标签页的显示方式
5. **状态管理**: 合理使用受控和非受控模式sidebar_position: 7

## 动态标签页

```jsx live
function DynamicTabsExample() {
  const [tabs, setTabs] = React.useState([
    { key: 'tab1', label: '标签 1', content: '这是标签 1 的内容' },
    { key: 'tab2', label: '标签 2', content: '这是标签 2 的内容' }
  ]);
  const [activeKey, setActiveKey] = React.useState('tab1');
  const [counter, setCounter] = React.useState(3);

  const addTab = () => {
    const newTab = {
      key: `tab${counter}`,
      label: `标签 ${counter}`,
      content: `这是动态添加的标签 ${counter} 的内容`
    };
    setTabs([...tabs, newTab]);
    setActiveKey(newTab.key);
    setCounter(counter + 1);
  };

  const removeTab = (targetKey) => {
    const newTabs = tabs.filter(tab => tab.key !== targetKey);
    setTabs(newTabs);
    
    if (activeKey === targetKey && newTabs.length > 0) {
      setActiveKey(newTabs[0].key);
    }
  };

  return (
    <div>
      <div style={{ marginBottom: '16px' }}>
        <button 
          onClick={addTab}
          style={{ 
            padding: '8px 16px', 
            border: 'none', 
            borderRadius: '4px', 
            background: '#10b981', 
            color: 'white',
            cursor: 'pointer',
            marginRight: '8px'
          }}
        >
          ➕ 添加标签页
        </button>
        <span style={{ color: '#6b7280' }}>
          当前有 {tabs.length} 个标签页
        </span>
      </div>

      <Tabs
        activeKey={activeKey}
        onChange={setActiveKey}
      >
        {tabs.map(tab => (
          <Tab 
            key={tab.key} 
            label={
              <span style={{ display: 'flex', alignItems: 'center', gap: '8px' }}>
                {tab.label}
                {tabs.length > 1 && (
                  <button
                    onClick={(e) => {
                      e.stopPropagation();
                      removeTab(tab.key);

                      background: 'none',
                      border: 'none',
                      color: '#ef4444',
                      cursor: 'pointer',
                      padding: '2px',
                      borderRadius: '2px',

```jsx
function DynamicTabs() {
  const [tabs, setTabs] = useState([
    { key: 'tab1', label: '标签 1', content: '内容 1' }
  ]);
  const [activeKey, setActiveKey] = useState('tab1');

  const addTab = () => {
    const newKey = `tab${Date.now()}`;
    setTabs([...tabs, { 
      key: newKey, 
      label: `新标签 ${tabs.length + 1}`, 
      content: `新内容 ${tabs.length + 1}` 
    }]);
    setActiveKey(newKey);
  };

  const removeTab = (targetKey) => {
    const newTabs = tabs.filter(tab => tab.key !== targetKey);
    setTabs(newTabs);
    if (activeKey === targetKey && newTabs.length > 0) {
      setActiveKey(newTabs[0].key);
    }
  };

  return (
    <div>
      <button onClick={addTab}>添加标签页</button>
      <Tabs activeKey={activeKey} onChange={setActiveKey}>
        {tabs.map(tab => (
          <Tab key={tab.key} label={tab.label}>
            <div>{tab.content}</div>
          </Tab>
        ))}
      </Tabs>
    </div>
  );
}

## API

### Tabs Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| activeKey | `string` | - | 当前激活的标签页 key（受控模式） |
| children | `React.ReactNode` | - | Tab 子组件 |
| centered | `boolean` | `false` | 标签页是否居中显示 |
| className | `string` | - | 自定义样式类名 |
| defaultActiveKey | `string` | - | 默认激活的标签页 key（非受控模式） |
| onChange | `(key: string) => void` | - | 标签页切换时的回调函数 |

### Tab Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| key | `string` | - | 标签页的唯一标识符（必填） |
| label | `React.ReactNode` | - | 标签页显示的标签文本或元素 |
| children | `React.ReactNode` | - | 标签页的内容 |

## 注意事项

1. **Key 的重要性**: 每个 `Tab` 必须有唯一的 `key` 属性
2. **受控 vs 非受控**: 使用 `activeKey` 为受控模式，使用 `defaultActiveKey` 为非受控模式
3. **内容渲染**: 只有当前激活的标签页内容会被渲染
4. **键盘导航**: 支持左右箭头键在标签间导航
5. **无障碍访问**: 自动生成正确的 ARIA 属性和角色

## 最佳实践

1. **标签数量**: 建议标签页数量不超过 7 个，过多会影响用户体验
2. **标签文本**: 保持标签文本简洁明了，避免过长
3. **内容组织**: 确保标签页之间的内容逻辑清晰
4. **响应式**: 在移动端考虑标签页的显示方式
5. **状态管理**: 合理使用受控和非受控模式sidebar_position: 7

```jsx live
function DynamicTabsExample() {
  const [tabs, setTabs] = React.useState([
    { key: 'tab1', label: '标签 1', content: '这是标签 1 的内容' },
    { key: 'tab2', label: '标签 2', content: '这是标签 2 的内容' }
  ]);
  const [activeKey, setActiveKey] = React.useState('tab1');
  const [counter, setCounter] = React.useState(3);

  const addTab = () => {
    const newTab = {
      key: `tab${counter}`,
      label: `标签 ${counter}`,
      content: `这是动态添加的标签 ${counter} 的内容`
    };
    setTabs([...tabs, newTab]);
    setActiveKey(newTab.key);
    setCounter(counter + 1);
  };

  const removeTab = (targetKey) => {
    const newTabs = tabs.filter(tab => tab.key !== targetKey);
    setTabs(newTabs);
    
    if (activeKey === targetKey && newTabs.length > 0) {
      setActiveKey(newTabs[0].key);
    }
  };

  return (
    <div>
      <div style={{ marginBottom: '16px' }}>
        <button 
          onClick={addTab}
          style={{

          当前有 {tabs.length} 个标签页
        </span>
      </div>

```jsx
function DynamicTabs() {
  const [tabs, setTabs] = useState([
    { key: 'tab1', label: '标签 1', content: '内容 1' }
  ]);
  const [activeKey, setActiveKey] = useState('tab1');

  const addTab = () => {
    const newKey = `tab${Date.now()}`;
    setTabs([...tabs, { 
      key: newKey, 
      label: `新标签 ${tabs.length + 1}`, 
      content: `新内容 ${tabs.length + 1}` 
    }]);
    setActiveKey(newKey);
  };

  const removeTab = (targetKey) => {
    const newTabs = tabs.filter(tab => tab.key !== targetKey);
    setTabs(newTabs);
    if (activeKey === targetKey && newTabs.length > 0) {
      setActiveKey(newTabs[0].key);
    }
  };

  return (
    <div>
      <button onClick={addTab}>添加标签页</button>
      <Tabs activeKey={activeKey} onChange={setActiveKey}>
        {tabs.map(tab => (
          <Tab key={tab.key} label={tab.label}>
            <div>{tab.content}</div>
          </Tab>
        ))}
      </Tabs>
    </div>
  );
}

## API

### Tabs Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| activeKey | `string` | - | 当前激活的标签页 key（受控模式） |
| children | `React.ReactNode` | - | Tab 子组件 |
| centered | `boolean` | `false` | 标签页是否居中显示 |
| className | `string` | - | 自定义样式类名 |
| defaultActiveKey | `string` | - | 默认激活的标签页 key（非受控模式） |
| onChange | `(key: string) => void` | - | 标签页切换时的回调函数 |

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| key | `string` | - | 标签页的唯一标识符（必填） |
| label | `React.ReactNode` | - | 标签页显示的标签文本或元素 |
| children | `React.ReactNode` | - | 标签页的内容 |

## 注意事项

1. **Key 的重要性**: 每个 `Tab` 必须有唯一的 `key` 属性
2. **受控 vs 非受控**: 使用 `activeKey` 为受控模式，使用 `defaultActiveKey` 为非受控模式
3. **内容渲染**: 只有当前激活的标签页内容会被渲染
4. **键盘导航**: 支持左右箭头键在标签间导航
5. **无障碍访问**: 自动生成正确的 ARIA 属性和角色

## 最佳实践

1. **标签数量**: 建议标签页数量不超过 7 个，过多会影响用户体验
2. **标签文本**: 保持标签文本简洁明了，避免过长
3. **内容组织**: 确保标签页之间的内容逻辑清晰
4. **响应式**: 在移动端考虑标签页的显示方式
5. **状态管理**: 合理使用受控和非受控模式
