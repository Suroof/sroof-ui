# Menu 菜单

导航菜单组件，支持多级菜单和各种交互模式。

## 基本用法

```jsx live
function MenuExample() {
  return (
    <Menu
      mode="horizontal"
      defaultSelectedKey="1"
      onSelect={(key) => console.log("选中:", key)}
    >
      <MenuItem itemKey="1">首页</MenuItem>
      <MenuItem itemKey="2">产品</MenuItem>
      <MenuItem itemKey="3">关于我们</MenuItem>
      <MenuItem itemKey="4">联系我们</MenuItem>
    </Menu>
  );
}
```

## 垂直菜单

```jsx live
function VerticalMenuExample() {
  return (
    <Menu mode="vertical" defaultSelectedKey="2">

      <MenuItem itemKey="2">导航二</MenuItem>
      <MenuItem itemKey="3">导航三</MenuItem>
      <MenuItem itemKey="4">导航四</MenuItem>

```

## 子菜单

```jsx live
function SubMenuExample() {
  return (
    <div style={{ width: 800, height: 150 }}>
      <Menu defaultSelectedKey="2-1">
        <MenuItem itemKey="1">首页</MenuItem>
        <SubMenu itemKey="2" title="产品">
          <MenuItem itemKey="2-1">产品A</MenuItem>
          <MenuItem itemKey="2-2">产品B</MenuItem>
          <MenuItem itemKey="2-3">产品C</MenuItem>
        </SubMenu>
        <MenuItem itemKey="3">关于我们</MenuItem>
      </Menu>
    </div>
  );
}
```

## 带图标的菜单

```jsx live
function IconMenuExample() {
  const HomeIcon = () => <span>🏠</span>;
  const UserIcon = () => <span>👤</span>;
  const SettingsIcon = () => <span>⚙️</span>;

  return (
    <div style={{ width: 800, height: 150 }}>
      <Menu defaultSelectedKey="1">
        <MenuItem itemKey="1" icon={<HomeIcon />}>
          首页
        </MenuItem>
        <MenuItem itemKey="2" icon={<UserIcon />}>
          用户
        </MenuItem>
        <SubMenu itemKey="3" title="设置" icon={<SettingsIcon />}>
          <MenuItem itemKey="3-1">基本设置</MenuItem>
          <MenuItem itemKey="3-2">高级设置</MenuItem>
        </SubMenu>
      </Menu>
    </div>
  );
}
```

## 受控模式

```jsx live
function ControlledMenuExample() {
  const [selectedKey, setSelectedKey] = React.useState("1");

  return (
    <div>
      <p>当前选中: {selectedKey}</p>
      <Menu selectedKey={selectedKey} onSelect={setSelectedKey}>
        <MenuItem itemKey="1">首页</MenuItem>
        <MenuItem itemKey="2">产品</MenuItem>
        <MenuItem itemKey="3">关于我们</MenuItem>
      </Menu>
    </div>
  );
}
```

## API

### Menu Props

| 属性               | 类型                                     | 默认值         | 说明               |
| ------------------ | ---------------------------------------- | -------------- | ------------------ |
| children           | `ReactNode`                              | -              | 菜单项内容         |
| mode               | `'horizontal' \| 'vertical' \| 'inline'` | `'horizontal'` | 菜单模式           |
| theme              | `'light' \| 'dark' \| 'glass'`           | `'light'`      | 菜单主题           |
| className          | `string`                                 | -              | 自定义类名         |
| defaultSelectedKey | `string`                                 | -              | 默认选中的菜单项   |
| selectedKey        | `string`                                 | -              | 受控选中的菜单项   |
| onSelect           | `(key: string) => void`                  | -              | 选择菜单项时的回调 |
| expandIcon         | `ReactNode`                              | -              | 自定义展开图标     |
| collapsible        | `boolean`                                | `false`        | 是否可折叠         |

### MenuItem Props

| 属性      | 类型         | 默认值  | 说明             |
| --------- | ------------ | ------- | ---------------- |
| itemKey   | `string`     | -       | 菜单项的唯一标识 |
| children  | `ReactNode`  | -       | 菜单项显示内容   |
| icon      | `ReactNode`  | -       | 菜单项图标       |
| disabled  | `boolean`    | `false` | 是否禁用         |
| danger    | `boolean`    | `false` | 是否为危险项     |
| className | `string`     | -       | 自定义类名       |
| onClick   | `() => void` | -       | 点击回调         |

### SubMenu Props

| 属性      | 类型        | 默认值  | 说明             |
| --------- | ----------- | ------- | ---------------- |
| itemKey   | `string`    | -       | 子菜单的唯一标识 |
| title     | `string`    | -       | 子菜单显示标题   |
| children  | `ReactNode` | -       | 子菜单项         |
| icon      | `ReactNode` | -       | 子菜单图标       |
| disabled  | `boolean`   | `false` | 是否禁用         |
| className | `string`    | -       | 自定义类名       |
