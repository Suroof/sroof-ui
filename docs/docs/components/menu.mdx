# Menu 菜单

导航菜单组件，支持多级菜单和各种交互模式。

## 基本用法

```jsx live
function MenuExample() {
  return (
    <Menu defaultSelectedKey="1" onSelect={(key) => console.log('选中:', key)}>
      <MenuItem key="1" label="首页" />
      <MenuItem key="2" label="产品" />
      <MenuItem key="3" label="关于我们" />
      <MenuItem key="4" label="联系我们" />
    </Menu>
  );
}
```

```jsx
import { Menu, MenuItem } from 'sroof-ui';

function Example() {
  return (
    <Menu defaultSelectedKey="1" onSelect={(key) => console.log('选中:', key)}>
      <MenuItem key="1" label="首页" />
      <MenuItem key="2" label="产品" />
      <MenuItem key="3" label="关于我们" />
      <MenuItem key="4" label="联系我们" />
    </Menu>
  );
}
```

## 垂直菜单

```jsx live
function VerticalMenuExample() {
  return (
    <Menu mode="vertical" defaultSelectedKey="2">
      <MenuItem key="1" label="导航一" />
      <MenuItem key="2" label="导航二" />
      <MenuItem key="3" label="导航三" />
      <MenuItem key="4" label="导航四" />
    </Menu>
  );
}
```

```jsx
<Menu mode="vertical" defaultSelectedKey="2">
  <MenuItem key="1" label="导航一" />
  <MenuItem key="2" label="导航二" />
  <MenuItem key="3" label="导航三" />
  <MenuItem key="4" label="导航四" />
</Menu>
```

## 子菜单

```jsx live
function SubMenuExample() {
  return (
    <Menu defaultSelectedKey="2-1">
      <MenuItem key="1" label="首页" />
      <SubMenu key="2" label="产品">
        <MenuItem key="2-1" label="产品A" />
        <MenuItem key="2-2" label="产品B" />
        <MenuItem key="2-3" label="产品C" />
      </SubMenu>
      <MenuItem key="3" label="关于我们" />
    </Menu>
  );
}
```

```jsx
import { Menu, MenuItem, SubMenu } from 'sroof-ui';

<Menu defaultSelectedKey="2-1">
  <MenuItem key="1" label="首页" />
  <SubMenu key="2" label="产品">
    <MenuItem key="2-1" label="产品A" />
    <MenuItem key="2-2" label="产品B" />
    <MenuItem key="2-3" label="产品C" />
  </SubMenu>
  <MenuItem key="3" label="关于我们" />
</Menu>
```

## 禁用项

```jsx live
function DisabledMenuExample() {
  return (
    <Menu defaultSelectedKey="1">
      <MenuItem key="1" label="可用项" />
      <MenuItem key="2" label="禁用项" disabled />
      <MenuItem key="3" label="另一个可用项" />
    </Menu>
  );
}
```

```jsx
<Menu defaultSelectedKey="1">
  <MenuItem key="1" label="可用项" />
  <MenuItem key="2" label="禁用项" disabled />
  <MenuItem key="3" label="另一个可用项" />
</Menu>
```

## API

### Menu Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| children | `ReactNode` | - | 菜单项内容 |
| mode | `'horizontal' \| 'vertical' \| 'inline'` | `'horizontal'` | 菜单模式 |
| defaultSelectedKey | `string` | - | 默认选中的菜单项 |
| onSelect | `(key: string) => void` | - | 选择菜单项时的回调 |
| theme | `'light' \| 'dark' \| 'glass'` | `'light'` | 菜单主题 |
| className | `string` | - | 自定义类名 |

### MenuItem Props

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| key | `string` | 菜单项的唯一标识 |
| label | `string` | 菜单项显示内容 |
| disabled | `boolean` | 是否禁用 |
| icon | `ReactNode` | 菜单项图标 |
| onClick | `() => void` | 点击回调 |

### SubMenu Props

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| key | `string` | 子菜单的唯一标识 |
| label | `string` | 子菜单显示内容 |
| children | `ReactNode` | 子菜单项 |
| disabled | `boolean` | 是否禁用 |
| icon | `ReactNode` | 子菜单图标 |
