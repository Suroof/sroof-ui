# Card 卡片

卡片组件，支持多种样式和状态。

## 基本用法

```jsx live
function ButtonExample() {
  return (
    <div style={{ display: "flex", gap: "8px", marginBottom: "16px" }}>
      <Card>卡片内容</Card>
    </div>
  );
}
```

```jsx live

  return (

```

## 边框显示

```jsx live
function ButtonLoadingExample() {
const [bordered, setBordered] = useState(true);
  const toggleBordered = () => {
    setBordered(!bordered); // 切换状态
  };
  return (
    <div style={{ display: "flex", gap: "8px", marginBottom: "16px" }}> 
      <Card  bordered={bordered} onClick={toggleBordered} clickable={true}>有无边框</Card>
    </div>
  );
}
```

## 阴影

## API

### Props

| 属性      | 类型                                                                           | 默认值      | 说明             |
| --------- | ------------------------------------------------------------------------------ | ----------- | ---------------- |
| title     | `string`                                                                       | -           | 卡片标题         |
| children  | `ReactNode`                                                                    | -           | 卡片内容         |
| className | `string`                                                                       | -           | 卡片样式         |
| size      | `'small' \| 'medium' \| 'large'`                                              | `'medium'`  | 卡片尺寸         |
| type      | `'default' \| 'primary' \| 'success' \| 'warning' \| 'danger' \| 'glass' \| 'gradient'` | `'default'` | 卡片类型         |
| clickable | `boolean`                                                                      | `false`     | 是否可点击       |
| onClick   | `() => void`                                                                   | -           | 点击事件         |
| bordered  | `boolean`                                                                      | `true`      | 是否显示边框     |
| shadow    | `'none' \| 'sm' \| 'md' \| 'lg' \| 'xl'`                                       | `'md'`      | 是否显示阴影     |
| icon      | `ReactNode`                                                                    | -           | 头部图标         |
| actions   | `ReactNode`                                                                    | -           | 操作按钮区域     |
| loading   | `boolean`                                                                      | `false`     | 是否加载中       |
