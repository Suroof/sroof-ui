# Input 输入框

输入框组件，支持多种样式和状态。

## 基本用法

```jsx live
function InputExample() {
  return (
    <div style={{ display: "flex", gap: "8px", marginBottom: "16px" }}>
      <Input size="medium" placeholder="默认输入框" />
    </div>
  );
}
```

## 输入框尺寸

```jsx live
function InputSizeExample() {
  return (
    <div
      style={{
        display: "flex",
        flexDirection: "column",
        gap: "12px",
        marginBottom: "16px",
      }}
    >
      <Input size="small" placeholder="小尺寸输入框" />
      <Input size="medium" placeholder="中等尺寸输入框" />
      <Input size="large" placeholder="大尺寸输入框" />
    </div>
  );
}
```

## 禁用状态

```jsx live
function InputDisabledExample() {
  return (
    <div
      style={{
        display: "flex",
        flexDirection: "column",
        gap: "12px",
        marginBottom: "16px",
      }}
    >
      <Input size="medium" placeholder="正常状态" />
      <Input size="medium" placeholder="禁用状态" disabled />
      <Input
        size="medium"
        placeholder="禁用状态带值"
        value="禁用的输入框"
        disabled
      />
    </div>
  );
}
```

## 受控组件

```jsx live
function InputControlledExample() {
  const [value, setValue] = React.useState("");

  return (
    <div style={{ marginBottom: "16px" }}>
      <Input
        value={value}
        size="medium"
        onChange={(e) => setValue(e.target.value)}
        placeholder="受控输入框"
      />
      <p style={{ marginTop: "8px", color: "#666" }}>当前值: {value}</p>
    </div>
  );
}
```

## API

### Props

| 属性         | 类型                                                 | 默认值     | 说明               |
| ------------ | ---------------------------------------------------- | ---------- | ------------------ |
| type         | `string`                                             | `'text'`   | 输入框类型         |
| placeholder  | `string`                                             | -          | 占位符文本         |
| size         | `'small' \| 'medium' \| 'large'`                     | `'medium'` | 输入框尺寸         |
| value        | `string`                                             | -          | 输入框的值（受控） |
| disabled     | `boolean`                                            | `false`    | 是否禁用           |
| onChange     | `(e: React.ChangeEvent<HTMLInputElement>) => void`   | -          | 值变化时的回调函数 |
| className    | `string`                                             | -          | 自定义类名         |
| style        | `React.CSSProperties`                                | -          | 自定义样式         |
| onFocus      | `(e: React.FocusEvent<HTMLInputElement>) => void`    | -          | 获得焦点时的回调   |
| onBlur       | `(e: React.FocusEvent<HTMLInputElement>) => void`    | -          | 失去焦点时的回调   |
| onKeyDown    | `(e: React.KeyboardEvent<HTMLInputElement>) => void` | -          | 按键按下时的回调   |
| onKeyUp      | `(e: React.KeyboardEvent<HTMLInputElement>) => void` | -          | 按键释放时的回调   |
| maxLength    | `number`                                             | -          | 最大输入长度       |
| minLength    | `number`                                             | -          | 最小输入长度       |
| readOnly     | `boolean`                                            | `false`    | 是否只读           |
| autoFocus    | `boolean`                                            | `false`    | 是否自动获得焦点   |
| autoComplete | `string`                                             | -          | 自动完成属性       |
