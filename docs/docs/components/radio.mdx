# Radio 单选框

单选框组件，支持受控和非受控模式。通常与 RadioGroup 组件配合使用。

## 基本用法 - 垂直布局

```jsx live
function RadioGroupVerticalExample() {
  return (
    <RadioGroup defaultValue="option2" name="vertical-group">
      <Radio value="option1">选项 1</Radio>
      <Radio value="option2">选项 2</Radio>
      <Radio value="option3">选项 3</Radio>
    </RadioGroup>
  );
}
```

## 单选框组 - 水平布局

```jsx live
function RadioGroupHorizontalExample() {
  return (
    <RadioGroup direction="horizontal" defaultValue="small" name="size-group">
      <Radio value="small">小</Radio>
      <Radio value="medium">中</Radio>
      <Radio value="large">大</Radio>
    </RadioGroup>
  );
}
```

## 受控模式

```jsx live
function RadioControlledExample() {
  const [value, setValue] = React.useState('react');
  
  return (
    <div>
      <RadioGroup 
        value={value} 
        onChange={setValue}
        name="framework-group"
      >
        <Radio value="react">React</Radio>
        <Radio value="vue">Vue</Radio>
        <Radio value="angular">Angular</Radio>
        <Radio value="svelte">Svelte</Radio>
      </RadioGroup>
      <p style={{ marginTop: '16px', color: '#666' }}>
        当前选择: {value}
      </p>
    </div>
  );
}
```

## 禁用状态

```jsx live
function RadioDisabledExample() {
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '12px' }}>
      <Radio value="normal">正常状态</Radio>
      <Radio value="disabled" disabled>
        禁用状态
      </Radio>
      <Radio value="disabled-checked" disabled defaultChecked>
        禁用且选中
      </Radio>
    </div>
  );
}
```

## 单选框组禁用

```jsx live
function RadioGroupDisabledExample() {
  return (
    <RadioGroup disabled defaultValue="option2" name="disabled-group">
      <Radio value="option1">选项 1</Radio>
      <Radio value="option2">选项 2</Radio>
      <Radio value="option3">选项 3</Radio>

```

## 混合禁用状态

```jsx live
function RadioMixedDisabledExample() {
  return (
    <RadioGroup defaultValue="enabled1" name="mixed-group">
      <Radio value="enabled1">可用选项 1</Radio>
      <Radio value="disabled1" disabled>禁用选项</Radio>
      <Radio value="enabled2">可用选项 2</Radio>
      <Radio value="disabled2" disabled>另一个禁用选项</Radio>
    </RadioGroup>
  );
}
```

## 自定义样式

```jsx live
function RadioCustomExample() {
  return (
    <RadioGroup 
      defaultValue="custom1" 
      name="custom-group"
      className="custom-radio-group"
    >
      <Radio value="custom1" className="custom-radio">
        自定义样式 1
      </Radio>
      <Radio value="custom2" className="custom-radio">
        自定义样式 2

  );

```

## API

### Radio Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| value | `string` | - | 选项值（必填） |
| children | `React.ReactNode` | - | 选项内容 |
| checked | `boolean` | - | 是否选中（受控模式） |
| defaultChecked | `boolean` | `false` | 默认选中（非受控模式） |
| disabled | `boolean` | `false` | 是否禁用 |
| name | `string` | - | 单选框组名称 |
| onChange | `(value: string) => void` | - | 值变化回调 |
| className | `string` | - | 自定义类名 |

### RadioGroup Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| value | `string` | - | 单选框组的值（受控模式） |
| defaultValue | `string` | - | 默认选中的值（非受控模式） |
| name | `string` | - | 单选框组名称 |
| onChange | `(value: string) => void` | - | 值变化回调 |
| disabled | `boolean` | `false` | 是否禁用整个组 |
| direction | `'horizontal' \| 'vertical'` | `'vertical'` | 布局方向 |
| className | `string` | - | 自定义类名 |
| children | `React.ReactNode` | - | 子元素 |

## 注意事项

1. **受控 vs 非受控**：使用 `checked` 属性时为受控模式，使用 `defaultChecked` 为非受控模式
2. **RadioGroup 优势**：推荐使用 `RadioGroup` 来管理多个 `Radio`，它会自动处理互斥逻辑
3. **name 属性**：在 `RadioGroup` 中会自动为所有子 `Radio` 设置相同的 `name`
4. **键盘导航**：支持方向键在选项间导航
5. **无障碍访问**：自动生成唯一 ID 和正确的 ARIA 属性
