# Skeleton 骨架屏

在需要等待加载内容的位置提供一个占位图形组合。

## 基本用法

```jsx live
function SkeletonExample() {
  const [loading, setLoading] = React.useState(true);
  
  return (
    <div>
      <Button 
        onClick={() => setLoading(!loading)}
        style={{ marginBottom: '16px' }}
      >
        {loading ? '显示内容' : '显示骨架屏'}
      </Button>
      
      {loading ? (
        <Skeleton />
      ) : (
        <div>
          <h3>实际内容标题</h3>
          <p>这里是实际的内容文本，当数据加载完成后会显示这些内容。</p>
        </div>
      )}
    </div>
  );
}
```

```jsx live
            <p style={{ margin: '0' }}>这是第四段内容描述。</p>
```

```jsx live
}
```

## 列表骨架屏

```jsx live
function SkeletonListExample() {
  const [loading, setLoading] = React.useState(true);
  
  const listData = [
    { title: '第一项标题', description: '这是第一项的描述内容' },
    { title: '第二项标题', description: '这是第二项的描述内容' },
    { title: '第三项标题', description: '这是第三项的描述内容' }
  ];
  
  return (
    <div>
      <Button 
        onClick={() => setLoading(!loading)}
        style={{ marginBottom: '16px' }}
      >
        {loading ? '加载数据' : '显示骨架屏'}
      </Button>
      
      <div style={{ display: 'flex', flexDirection: 'column', gap: '16px' }}>
        {loading ? (
          Array.from({ length: 3 }).map((_, index) => (
            <Skeleton 
              key={index}
              avatar 
              active
              paragraph={{ rows: 1 }}
            />
          ))
        ) : (
          listData.map((item, index) => (
            <div key={index} style={{ 
              display: 'flex', 
              alignItems: 'center', 
              gap: '12px',
              padding: '12px',
              border: '1px solid #f0f0f0',
              borderRadius: '6px'
            }}>
              <div style={{ 
                width: '40px', 
                height: '40px', 
                borderRadius: '50%', 
                backgroundColor: '#1890ff',
                display: 'flex',
                alignItems: 'center',
                justifyContent: 'center',
                color: 'white'
              }}>
                {index + 1}
              </div>
              <div>
                <h4 style={{ margin: '0 0 4px 0' }}>{item.title}</h4>
                <p style={{ margin: '0', color: '#666' }}>{item.description}</p>
              </div>
            </div>
          ))
        )}
      </div>
    </div>
  );
}
```

## API

### Skeleton Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| active | 是否展示动画效果 | `boolean` | `false` |
| avatar | 是否显示头像占位图 | `boolean \| SkeletonAvatarProps` | `false` |
| className | 自定义类名 | `string` | - |
| loading | 为 true 时，显示占位图。反之则直接展示子组件 | `boolean` | - |
| paragraph | 是否显示段落占位图 | `boolean \| SkeletonParagraphProps` | `true` |
| round | 为 true 时，段落和标题显示圆角 | `boolean` | `false` |
| style | 自定义样式 | `React.CSSProperties` | - |
| title | 是否显示标题占位图 | `boolean \| SkeletonTitleProps` | `true` |

### SkeletonAvatarProps

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| active | 是否展示动画效果 | `boolean` | `false` |
| shape | 指定头像的形状 | `'circle' \| 'square'` | `'circle'` |
| size | 设置头像占位图的大小 | `'large' \| 'small' \| 'default' \| number` | `'default'` |

### SkeletonTitleProps

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| width | 设置标题占位图的宽度 | `number \| string` | - |

### SkeletonParagraphProps

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| rows | 设置段落占位图的行数 | `number` | - |
| width | 设置段落占位图的宽度，若为数组时则为对应的每行宽度，反之则是最后一行的宽度 | `number \| string \| Array<number \| string>` | - |

### SkeletonButton Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| active | 是否展示动画效果 | `boolean` | `false` |
| block | 将按钮宽度调整为其父宽度的选项 | `boolean` | `false` |
| className | 自定义类名 | `string` | - |
| shape | 指定按钮的形状 | `'circle' \| 'round' \| 'default'` | `'default'` |
| size | 设置按钮的大小 | `'large' \| 'small' \| 'default'` | `'default'` |
| style | 自定义样式 | `React.CSSProperties` | - |

### SkeletonInput Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| active | 是否展示动画效果 | `boolean` | `false` |
| className | 自定义类名 | `string` | - |
| size | 设置输入框的大小 | `'large' \| 'small' \| 'default'` | `'default'` |
| style | 自定义样式 | `React.CSSProperties` | - |

### SkeletonImage Props

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| active | 是否展示动画效果 | `boolean` | `false` |
| className | 自定义类名 | `string` | - |
| style | 自定义样式 | `React.CSSProperties` | - |
