# Form 表单

表单组件，用于数据收集、校验和提交。

## 基本用法

```jsx live
function FormExample() {
  return (
    <Form onSubmit={(e) => console.log("Form submitted", e)}>
      <FormItem label="用户名" required>
        <Input placeholder="请输入用户名" />
      </FormItem>
      <FormItem label="邮箱" required>
        <Input type="email" placeholder="请输入邮箱" />
      </FormItem>
      <FormItem label="密码" required>
        <Input type="password" placeholder="请输入密码" />
      </FormItem>
      <FormActions align="center">
        <Button variant="outline">取消</Button>
        <Button variant="primary" type="submit">
          提交
        </Button>
      </FormActions>
    </Form>
  );
}
```

## 表单验证

```jsx live
function FormValidationExample() {
  return (
    <Form>
      <FormItem 
        label="邮箱" 
        required 
        rules={[
          { required: true, message: '请输入邮箱' },
          { type: 'email', message: '请输入正确的邮箱格式' }
        ]}
      >
        <Input placeholder="请输入邮箱" />
      </FormItem>
      <FormItem 
        label="密码" 
        required
        rules={[
          { required: true, message: '请输入密码' },
          { min: 6, message: '密码至少6位' }
        ]}
      >
        <Input type="password" placeholder="请输入密码" />
      </FormItem>
      <FormActions>
        <Button variant="primary" type="submit">提交</Button>
      </FormActions>
    </Form>
  );
}
```

## 加载状态

```jsx live
function FormLoadingExample() {
  const [loading, setLoading] = React.useState(false);
  
  const handleSubmit = () => {
    setLoading(true);
    setTimeout(() => setLoading(false), 2000);
  };
  
  return (
    <Form onSubmit={handleSubmit}>
      <FormItem label="用户名">
        <Input placeholder="请输入用户名" />
      </FormItem>
      <FormActions>
        <Button variant="primary" type="submit" loading={loading}>
          {loading ? '提交中...' : '提交'}
        </Button>
      </FormActions>
    </Form>
  );
}
```

## API

### Form Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| layout | `'vertical' \| 'horizontal' \| 'inline'` | `'vertical'` | 表单布局 |
| disabled | `boolean` | `false` | 是否禁用整个表单 |
| onSubmit | `(values: any) => void` | - | 表单提交回调 |
| onValuesChange | `(changedValues: any, allValues: any) => void` | - | 表单值变化回调 |
| initialValues | `object` | - | 表单初始值 |
| className | `string` | - | 自定义类名 |

### FormItem Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| label | `string` | - | 标签文本 |
| name | `string` | - | 字段名 |
| required | `boolean` | `false` | 是否必填 |
| rules | `Rule[]` | - | 验证规则 |
| help | `string` | - | 提示信息 |
| validateStatus | `'success' \| 'warning' \| 'error' \| 'validating'` | - | 校验状态 |
| children | `ReactNode` | - | 表单控件 |
| className | `string` | - | 自定义类名 |

### FormActions Props

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| align | `'left' \| 'center' \| 'right'` | `'left'` | 对齐方式 |
| children | `ReactNode` | - | 操作按钮 |
| className | `string` | - | 自定义类名 |
