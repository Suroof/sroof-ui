# LineMotion 线条动图

线条动图组件，支持自定义 svg 绘图

## 基本用法

```jsx live
function Example() {
  return (
    <div>
      <LineMotion
        pathData="M0,0 L100,100 L200,50 L300,150 L400,100"
        duration={3}
        ease="power1.inOut"
        stroke="#3b82f6"
        strokeWidth={3}
      />
    </div>
  );
}
```

## 平滑曲线

```jsx live
function Example() {
  return (
    <div>
      <LineMotion
        pathData="M10,100 C100,20 200,180 390,100"
        duration={3}
        ease="power1.inOut"
        stroke="#3b82f6"
        strokeWidth={3}
      />
    </div>
  );
}
```

## API

### Props

| 属性        | 类型   | 默认值         | 说明                                                                                         |
| ----------- | ------ | -------------- | -------------------------------------------------------------------------------------------- |
| pathData    | string | -              | SVG 路径数据字符串，定义要绘制的路径形状，例如: "M10 10 L20 20" 表示从(10,10)到(20,20)的直线 |
| duration    | number | 2              | 动画持续时间（秒）                                                                           |
| ease        | string | "power1.inOut" | 动画缓动函数，可选值参考 GSAP 的 ease 选项                                                   |
| stroke      | string | "black"        | 路径描边颜色                                                                                 |
| strokeWidth | number | 3              | 路径描边宽度                                                                                 |
| className   | string | -              | 可选的 CSS 类名，用于自定义样式                                                              |
| id          | string | -              | 可选的元素 ID                                                                                |
