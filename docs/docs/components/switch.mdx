# Switch 开关

开关组件用于在两个状态之间进行切换，常用于设置选项的开启或关闭。

## 基本用法

最简单的开关使用方式：

```jsx live
function BasicSwitch() {
  return <Switch defaultChecked />;
}
```

## 不同尺寸
```jsx live
function SwitchSizes() {
  return (
    <div style={{ display: 'flex', alignItems: 'center', gap: '20px' }}>
      <Switch size="small" defaultChecked />
      <span>小尺寸</span>
      
      <Switch size="medium" defaultChecked />
      <span>中等尺寸</span>
      
      <Switch size="large" defaultChecked />
      <span>大尺寸</span>
    </div>
  );
}
```

## 带标签的开关
```jsx live
function SwitchWithLabels() {

      <Switch defaultChecked>启用通知</Switch>
      
      <Switch disabled defaultChecked>禁用状态</Switch>
    </div>

```

## 受控组件
```jsx live
function ControlledSwitch() {
  const [checked, setChecked] = React.useState(false);
  
  return (
    <div>
      <p>当前状态: {checked ? '开启' : '关闭'}</p>
      <Switch 
        checked={checked} 
        onChange={setChecked}
      >
        受控开关
      </Switch>
    </div>
  );
}
```

## 加载状态
```jsx live
function LoadingSwitch() {
  const [loading, setLoading] = React.useState(false);
  const [checked, setChecked] = React.useState(false);
  
  const handleChange = (newChecked) => {
    setLoading(true);
    // 模拟异步操作
    setTimeout(() => {
      setChecked(newChecked);
      setLoading(false);
    }, 1000);
  };
  
  return (
    <Switch 
      checked={checked}
      onChange={handleChange}
      loading={loading}
    >
      保存设置
    </Switch>
  );
}
```

## 禁用状态
```jsx live
function DisabledSwitch() {
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '15px' }}>
      <Switch disabled>禁用（默认关闭）</Switch>
      <Switch disabled defaultChecked>禁用（默认开启）</Switch>
    </div>
  );
}
```

## 实际应用场景
```jsx live
function SettingsPanel() {
  const [notifications, setNotifications] = React.useState(true);
  const [darkMode, setDarkMode] = React.useState(false);
  const [autoSave, setAutoSave] = React.useState(true);
  
  return (
    <div style={{ 
      maxWidth: '400px', 
      padding: '20px', 
      border: '1px solid #e5e7eb', 
      borderRadius: '8px' 
    }}>
      <h3 style={{ margin: '0 0 15px 0' }}>系统设置</h3>
      
      <div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center', marginBottom: '15px' }}>
        <span>启用通知</span>
        <Switch 
          checked={notifications} 
          onChange={setNotifications} 
        />
      </div>
      
      <div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center', marginBottom: '15px' }}>
        <span>深色模式</span>
        <Switch 
          checked={darkMode} 
          onChange={setDarkMode} 
          color="#9c27b0"
        />
      </div>
      
      <div style={{ display: 'flex', justifyContent: 'space-between', alignItems: 'center' }}>
        <span>自动保存</span>
        <Switch 
          checked={autoSave} 
          onChange={setAutoSave} 
        />
      </div>
    </div>
  );
}
```

## API

### SwitchProps

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| checked | boolean | - | 是否选中（受控模式） |
| defaultChecked | boolean | false | 默认选中（非受控模式） |
| disabled | boolean | false | 是否禁用 |
| onChange | (checked: boolean) => void | - | 值变化回调函数 |
| className | string | - | 自定义类名 |
| size | 'small' \| 'medium' \| 'large' | 'medium' | 开关尺寸 |
| children | React.ReactNode | - | 开关标签 |
| labelPosition | 'left' \| 'right' | 'right' | 标签位置 |
| loading | boolean | false | 加载状态 |
| color | string | - | 自定义颜色 |
